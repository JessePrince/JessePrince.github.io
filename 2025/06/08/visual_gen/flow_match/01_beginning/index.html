1 The Definition of the Velocity ODE

Let’s consider a particle moving in the space, we use function ψt(x)\psi_t(x)ψt​(x) to describe its current position at time t∈[0,1]t\in[0,1]t∈[0,1] starting from xxx. That says, given the initial position xxx and the time ttt, the function ψ:[0,1]×Rd→Rd\psi:[0,1]\times \mathbb{R}^d\rightarrow \mathbb{R}^dψ:[0,1]×Rd→Rd tells us the current position.

Now we define another function ut(x)u_t(x)ut​(x), which is a vector field u:[0,1]×Rd→Rdu:[0,1]\times \mathbb{R}^d \rightarrow \mathbb{R}^du:[0,1]×Rd→Rd, this vector field describes the velocity of the particle. Given the current position xxx and time ttt, it tells us the velocity of the particle.

We now can define an ODE describing the movement of the particle:

ddtψt(x)=ut(ψt(x))(1)\frac{\mathrm{d}}{\mathrm{d}t} \psi_t(x) = u_t(\psi_t(x))\tag{1} dtd​ψt​(x)=ut​(ψt​(x))(1)

Basically, this equation says, the derivative of the position function ψt\psi_tψt​ is the velocity of the particle given by the vector field function utu_tut​. We define that ψ0(x)=x\psi_0(x)=xψ0​(x)=x, since at t=0t=0t=0 the particle is at the initial position.

Now, if we know the velocity field utu_tut​, solving the ODE will generate a path flowing from ψ0\psi_0ψ0​ to ψ1\psi_1ψ1​, we say the velocity field utu_tut​ generates the probability path ptp_tpt​ if its flow ψt\psi_tψt​ satisfies

Xt:=ψt(X0)∼pt  for  X0∼p0X_t := \psi_t(X_0)\sim p_t \ \ \text{for}\ \ X_0\sim p_0 Xt​:=ψt​(X0​)∼pt​  for  X0​∼p0​

Therefore, if we have the velocity field, solving the ODE provides us X1=ψ1(X0)X_1=\psi_1(X_0)X1​=ψ1​(X0​), let’s say X1X_1X1​ is from target distribution qqq and X0X_0X0​ is from the source distribution ppp, then p0=pp_0=pp0​=p and p1=qp_1=qp1​=q, Flow Matching is trying to learn the field utθu_t^\thetautθ​ such that its flow ψt\psi_tψt​ generates such a probability path.

2 Flow Matching Objective

Let’s start from a standard example, let the source distribution be a standard Gaussian p:=p0=N(x∣0,I)p:=p_0=\mathcal{N}(x|0,I)p:=p0​=N(x∣0,I), and the probability path ptp_tpt​ is the aggregation of the conditioned probability path pt∣1(x∣x1)p_{t|1}(x|x_1)pt∣1​(x∣x1​), where x1x_1x1​ is from the training dataset, let’s take the aggregation:

pt(x)=∫pt∣1(x∣x1)q(x1)dx1pt∣1(x∣x1)=N(x∣tx1,(1−t)2I)p_t(x) = \int p_{t|1}(x|x_1)q(x_1)dx_1\quad p_{t|1}(x|x_1)=\mathcal{N}(x|tx_1,(1-t)^2I) pt​(x)=∫pt∣1​(x∣x1​)q(x1​)dx1​pt∣1​(x∣x1​)=N(x∣tx1​,(1−t)2I)

This is just the marginal probability calculation, this path is also known as the conditional optimal-transport or linear path. Using this path, we may say Xt∼ptX_t\sim p_tXt​∼pt​ is the linear combination of X0∼pX_0\sim pX0​∼p and X1∼qX_1\sim qX1​∼q, note that here the X0X_0X0​ and X1X_1X1​ are random variables drawn from distributions.

Xt=[tX1+(1−t)X0]∼pt(2)X_t = [tX_1 + (1-t)X_0]\sim p_t\tag{2} Xt​=[tX1​+(1−t)X0​]∼pt​(2)

The flow matching loss is straightforward

LFM=Et,Xt∥utθ(Xt)−ut(Xt)∥2t∼U[0,1]  and  Xt∼pt\mathcal{L}_{FM} = \mathbb{E}_{t,X_t}\|u_t^\theta(X_t)-u_t(X_t)\|^2\quad t\sim \mathcal{U}[0,1]\ \ \text{and}\ \ X_t\sim p_t LFM​=Et,Xt​​∥utθ​(Xt​)−ut​(Xt​)∥2t∼U[0,1]  and  Xt​∼pt​

The above loss function is intractable, but this objective simplifies drastically if we condition the loss on a single target example X1=x1X_1=x_1X1​=x1​ picked randomly from training set.

From Equation (2) let’s write

Xt∣1=tx1+(1−t)X0  ∼  pt∣1(⋅∣x1)=N(⋅∣tx1,(1−t)2I)(3)X_{t|1} = tx_1 + (1-t)X_0\ \ \sim\ \ p_{t|1}(\cdot|x_1) = \mathcal{N}(\cdot|tx_1,(1-t)^2I)\tag{3} Xt∣1​=tx1​+(1−t)X0​  ∼  pt∣1​(⋅∣x1​)=N(⋅∣tx1​,(1−t)2I)(3)

Now x1x_1x1​ is a specific sample from training dataset, let’s go back to Equation (1) and solve

ddtXt∣1=ut(Xt∣1∣x1)⇒x1−X0=ut(x∣x1)(4)\frac{\mathrm{d}}{\mathrm{d} t} X_{t|1}=u_t(X_{t|1}|x_1)\Rightarrow x_1-X_0 = u_t(x|x_1) \tag{4} dtd​Xt∣1​=ut​(Xt∣1​∣x1​)⇒x1​−X0​=ut​(x∣x1​)(4)

Rearrange Equation (3) gives us

X0=Xt∣1−tx11−t(5)X_0 = \frac{X_{t|1}-tx_1}{1-t}\tag{5} X0​=1−tXt∣1​−tx1​​(5)

Take (5) back to (4) yields

ut(x∣x1)=x1−x1−t(6)u_t(x|x_1) = \frac{x_1-x}{1-t} \tag{6} ut​(x∣x1​)=1−tx1​−x​(6)

this is the conditional vector field.

Now let’s take (6) into our objective, we obtain the conditional Flow Matching loss

LCFM=Et,Xt,X1∥utθ(Xt)−ut(Xt∣X1)∥2=Et,Xt,X1∥utθ(Xt)−X1−Xt1−t∥2=Equ.(2)Et,Xt,X1∥utθ(Xt)−X1−tX1−(1−t)X01−t∥2=Et,Xt,X1∥utθ(Xt)−(X1−X0)∥2\begin{aligned} \mathcal{L}_{CFM} &= \mathbb{E}_{t,X_t,X_1}\|u_t^\theta(X_t)-u_t(X_t|X_1)\|^2 \\ &=\mathbb{E}_{t,X_t,X_1}\left\|u_t^\theta(X_t) - \frac{X_1-X_t}{1-t}\right\|^2 \\ &\overset{\text{Equ.(2)}}{=} \mathbb{E}_{t,X_t,X_1}\left\|u_t^\theta(X_t) - \frac{X_1-tX_1-(1-t)X_0}{1-t}\right\|^2\\ &=\mathbb{E}_{t,X_t,X_1}\|u_t^\theta(X_t) - (X_1-X_0)\|^2 \end{aligned} LCFM​​=Et,Xt​,X1​​∥utθ​(Xt​)−ut​(Xt​∣X1​)∥2=Et,Xt​,X1​​∥∥∥∥∥​utθ​(Xt​)−1−tX1​−Xt​​∥∥∥∥∥​2=Equ.(2)Et,Xt​,X1​​∥∥∥∥∥​utθ​(Xt​)−1−tX1​−tX1​−(1−t)X0​​∥∥∥∥∥​2=Et,Xt​,X1​​∥utθ​(Xt​)−(X1​−X0​)∥2​

where t∼U[0,1]t\sim U[0,1]t∼U[0,1], X0∼N(0,I)X_0\sim \mathcal{N}(0,I)X0​∼N(0,I) and X1∼qX_1\sim qX1​∼q.

This is the simplest implementation of Flow Matching using optimal transportation and Gaussian source.
