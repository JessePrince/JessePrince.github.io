典型的环节与系统方框图

1 典型环节

所谓环节就是具有相同形式传递函数的原部件的分类。这些传递函数是控制系统最经典最典型的。

1.1 比例环节

传递函数

G(s)=KG(s) = K G(s)=K

这个系统就是将输入乘以一个常数K输出，所以叫做比例环节。典型的比例环节部件就是电位计，根据电阻指针的位置有对应比例的输出。

1.2 微分环节

传递函数

G(s)=sG(s) = s G(s)=s

不难发现，这就是Laplace变换微分定理，将输入乘以s，就是在时域微分。

1.3 积分环节

传递函数

G(s)=1sG(s) = \frac{1}{s} G(s)=s1​

这也是Laplace变换的积分定理，将输入除以s，就是在时域积分。

1.4 惯性环节

传递函数

G(s)=1Ts+1G(s) = \frac{1}{Ts+1} G(s)=Ts+11​

这个函数其实描述了一个缓慢消失的过程，我们把它的时域函数写出来。

h(t)=1−e−tTh(t) = 1-e^{-\frac{t}{T}} h(t)=1−e−Tt​

随着时间增加，函数值按指数减少，就像是因为惯性而保持一段距离，最后消失，其中TTT就是时间常数。

1.5 震荡环节

传递函数

G(s)=1T2s2+2ξTs+1G(s) = \frac{1}{T^2s^2+2\xi Ts+1} G(s)=T2s2+2ξTs+11​

函数描述的是一种震荡的状态，这个震荡可能是终值发散震荡，可能是一直震荡(也叫做临界震荡)，可能是种植收敛的震荡，这取决于阻尼系数ξ\xiξ。

1.6 一阶复合微分环节

传递函数

τs+1\tau s+1 τs+1

暂不做介绍

1.7 二阶复合微分环节

传递函数

τ2s2+2ξτs+1\tau^2s^2+2\xi \tau s+1 τ2s2+2ξτs+1

暂不做介绍。

1.8总结

任何的传递函数都可以看作典型环节的组合，这些典型环节可以构成多种多样的系统。

2 负载效应问题

现考虑两个电路连接在一起，第一级电路输入uru_rur​，输出uau_aua​，第二级电路输入uau_aua​，输出ucu_cuc​。但我们都知道，简单的耦合两个电路，第二级电路实际上会对第一级电路造成影响，这就叫做负载效应。对系统接入负载的时候，可能会导致系统不是简单的输入输出关系。

3 系统方框图

描述控制系统的一大方式就是使用方框图，用有向线段表示信号的流通方向，方框表示系统模块，圆形表示加法器。方框图指明了信号的流动以及各部分对于信号的处理。

有了系统的方框图，下一步就是求解整个系统的传递函数。

4 传递函数求解

4.1 消解系数法

把方框图所有的信号写出来，根据信号的处理把模块输入和输出的关系写出来，然后把中间变量全部消除，的到最终输出与输入的关系。这种方法最简单，但在系统变得很复杂的时候不简便。

4.2 结构图等效变换法

结构图有很多等效变换的法则，灵活使用它们可以将系统的传递函数找出来，这个过程也叫做化简。

4.2.1 反馈等效

系统前向模块是G(s)G(s)G(s)，反馈模块是H(s)H(s)H(s)，系统等效为

G(s)1+S(s)H(s)\frac{G(s)}{1+S(s)H(s)} 1+S(s)H(s)G(s)​

4.2.2 串联等效

系统G1(s)G_1(s)G1​(s)与G2(s)G_2(s)G2​(s)串联，等效为

G1(s)⋅G2(s)G_1(s)\cdot G_2(s) G1​(s)⋅G2​(s)

4.2.3 并联等效

系统G1(s)G_1(s)G1​(s)与G2(s)G_2(s)G2​(s)并联，等效为

G1(s)+G2(s)G_1(s)+G_2(s) G1​(s)+G2​(s)

4.3.4 比较点前移

加法器也叫比较器，因为它将两个输入的信号进行相减(信号1加上负的信号2)

比较点在G(s)G(s)G(s)后面，相当于没有经过G(s)G(s)G(s)，前移过后信号就经过G(s)G(s)G(s)了，这时候需要乘以逆系统1G(s)\frac{1}{G(s)}G(s)1​。

4.3.5 比较点后移

反过来，比较点本来在G(s)G(s)G(s)前面，相当于经过了G(s)G(s)G(s)，后移之后就没有经过G(s)G(s)G(s)了，所以要补上一个G(s)G(s)G(s)。

4.3.6 引出点前移

引出点在系统后面，则原信号是经过了系统的，移到前面之后就相当于没有经过系统就引出了，则需要补上一个G(s)G(s)G(s)

4.3.7 引出点后移

引出点在系统前面，则原信号没有经过系统，移到后面之后就相当于经过了系统才引出，所以需要乘以一个逆系统1G(s)\frac{1}{G(s)}G(s)1​
