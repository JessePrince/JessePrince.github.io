03 信号的统计检测理论

1 信号检测的基本模型

为了简便，从二元信号的检测模型开始说起。

模型的第一个模块是信源，信源是信号发出的地方，它在某一时刻发出一种信号，然后在另一时刻可能发送另一种信号。考虑到我们研究二元的情况，所以信源发送的信号只有两种情况，此处记为H0H_0H0​和H1H_1H1​。

但由于我们在接收的时候并不知道信源发送的是哪一个信号，所以H0H_0H0​和H1H_1H1​又被称作假设。

比如一个输出0和1的信源，输出0我们可以用假设H0H_0H0​表示，输出1可以用H1H_1H1​表示。

模型的第二部分是概率转移机构，这一部分实际上代表了各种干扰信号的因素。如果在理想信道中传输，我们可以构造一个判决准则，使得信号能被准确的判断出来。例如使用频率调制时，我们可以准确的根据频率的不同来判决信号。但是当信号受到噪声干扰的时候，原本应该接收到H0H_0H0​的信号有可能会被检测成H1H_1H1​，例如我们假设频率为200Hz的信号代表0，但是因为干扰，我们可能检测成了1，这时候H0H_0H0​被转移到了H1H_1H1​，但因为噪声是随机的，所以不一定H0H_0H0​随时都会被转移到H1H_1H1​，所以说概率转移机构是把噪声干扰背景中的假设Hj(j=0,1)H_j(j=0,1)Hj​(j=0,1)为真的信号以一定概率关系映射到观测空间中。

对于二元信号的例子，我们最终观测到的是信号是0还是1，所以是一个一维的问题，该模型的观测空间就是由一维随机观测信号(x∣Hj)(x|H_j)(x∣Hj​)组成的，即在假设HjH_jHj​为真的前提下，观测到的结果是xxx。如果是N维的观测结果，即进行了N次观测，则空间是N维的，观测结果为x=(x1,x2,…,xn)T\bold x = (x_1,x_2,\dots,x_n)^Tx=(x1​,x2​,…,xn​)T，同样N维观测空间可表示为(x∣Hj)(\bold x|H_j)(x∣Hj​)。

模型的第四部分是判决规则，在观测量落入观测空间后，通过准则来判断信号到底是哪一个。

所以统计判决的任务，就是去划分观测空间的任务。假设观测的信号落在观测空间RRR中，通过划分得到子空间R1R_1R1​和R2R_2R2​，那么落在R1R_1R1​的信号我们就认为是H0H_0H0​，落在R2R_2R2​的信号我们就认为是H1H_1H1​。而且划分出的空间应该满足R=R1∪R2R = R_1\cup R_2R=R1​∪R2​，R1∩R2=ϕR_1\cap R_2=\phiR1​∩R2​=ϕ。因为划分的结果不能还有观测空间没有被定义。子空间R1R_1R1​, R2R_2R2​也叫做判决域。

如果是有M个观测信号，那么空间就会被划分成MMM个子空间，每一个子空间代表着一种判决结果。

2 二元信号的判决概率

对于二元信号落在观测空间的情况，一共有四种

  1. 假设是0，判决为0(H0∣H0)(H_0|H_0)(H0​∣H0​)
  2. 假设是0，判决为1(H1∣H0)(H_1|H_0)(H1​∣H0​)
  3. 假设是1，判决为0(H0∣H1)(H_0|H_1)(H0​∣H1​)
  4. 假设是1，判决为1(H1∣H1)(H_1|H_1)(H1​∣H1​)

其中两种是错的，两种是正确的，当假设和判决结果是正确的时候才是正确的结果。

由于噪声和信号等带来的随机性，所以判决应该是以一定概率去判决。所以对于每一种判决结果(Hi∣Hj)(H_i|H_j)(Hi​∣Hj​)，有响应的判决概率P(Hi∣Hj)P(H_i|H_j)P(Hi​∣Hj​)，它的含义是：在假设HjH_jHj​为真的前提下，判决假设HiH_iHi​成立的概率，对应的概率密度函数写成p(x∣Hj)p(x|H_j)p(x∣Hj​)，所以P(Hi∣Hj)P(H_i|H_j)P(Hi​∣Hj​)的概率是

P(Hi∣Hj)=∫Rip(x∣Hj)dxP(H_i|H_j) = \int_{R_i}p(x|H_j)dx P(Hi​∣Hj​)=∫Ri​​p(x∣Hj​)dx

因为判决是根据空间的划分来进行的，所以落在子空间RiR_iRi​时我们判决成HiH_iHi​，那么对于一个观测到的信号来说，判决成HiH_iHi​的概率就是用概率密度函数在RiR_iRi​空间上积分。

对于判决准则的确定，应该让正确判断的概率尽可能大，让错误判决的概率尽可能小。当我们划分空间的时候会同时影响四个可能判决的概率，所以判决域的划分应该存在一种最佳的方法。

3 贝叶斯准则

3.1 平均代价

为了最优化的划分观测空间，应该有一个度量来代表判决的性能，在这里，我们人为的去规定一个代价因子：

cijc_{ij} cij​

代表假设HjH_jHj​为真时，判决假设HiH_iHi​成立所付出的代价。也就是说，我们人为让判决器做出的判决都带上一个代价，来衡量它的判决性能。当然，判决正确的时候代价应该小，判决错误的时候代价应该大，所以

c10>c00,c01>c11c_{10} > c_{00}, c_{01}>c_{11} c10​>c00​,c01​>c11​

那么就可以写出在假设HjH_jHj​为真时判决付出的条件平均代价

C(Hj)=∑i=01cijP(Hi∣Hj)C(H_j) = \sum_{i=0}^1 c_{ij}P(H_i|H_j) C(Hj​)=i=0∑1​cij​P(Hi​∣Hj​)

有些时候我们是知道先验概率的，比如原信号是一个标准的周期矩形波，则0和1出现的概率各占一半，P(H0)=P(H1)=1/2P(H_0) = P(H_1)=1/2P(H0​)=P(H1​)=1/2。那么假设HjH_jHj​出现的先验概率是P(Hj)P(H_j)P(Hj​)，那么判决付出的总平均代价是

C=P(H0)C(H0)+P(H1)C(H1)C = P(H_0)C(H_0) + P(H_1)C(H_1) C=P(H0​)C(H0​)+P(H1​)C(H1​)

可以理解为通过先验概率对条件平均代价进行加权求和。

把这个求和展开来写，那么平均代价C就是

C=c00P(H0)∫R0p(x∣H0)dx+c10P(H0)∫R1p(x∣H0)dx+=c01P(H1)∫R0p(x∣H1)dx+c11P(H1)∫R1p(x∣H1)dx\begin{aligned} C = & c_{00}P(H_0)\int_{R_0}p(x|H_0)dx+c_{10}P(H_0)\int_{R_1}p(x|H_0)dx +\\ &=c_{01}P(H_1)\int_{R_0}p(x|H_1)dx+c_{11}P(H_1)\int_{R_1}p(x|H_1)dx \end{aligned} C=​c00​P(H0​)∫R0​​p(x∣H0​)dx+c10​P(H0​)∫R1​​p(x∣H0​)dx+=c01​P(H1​)∫R0​​p(x∣H1​)dx+c11​P(H1​)∫R1​​p(x∣H1​)dx​

由于观测空间的划分是满足R=R0∪R1R = R_0\cup R_1R=R0​∪R1​, R0∩R1=ϕR_0\cap R_1=\phiR0​∩R1​=ϕ,又根据概率密度函数的性质有

∫Rp(x∣Hj)dx=1\int_R p(x|H_j)dx=1 ∫R​p(x∣Hj​)dx=1

则代入到上面的式子

C=c10P(H0)+c11P(H1)+∫R0[(P(H1)(c01−c11)p(x∣H1))−(P(H0)(c10−c00)p(x∣H0))]dx\begin{aligned} C =& c_{10}P(H_0) + c_{11}P(H_1)+\int_{R_0}[(P(H_1)(c_{01}- c_{11})p(x|H_1))-\\ &(P(H_0)(c_{10}-c_{00})p(x|H_0))]dx \end{aligned} C=​c10​P(H0​)+c11​P(H1​)+∫R0​​[(P(H1​)(c01​−c11​)p(x∣H1​))−(P(H0​)(c10​−c00​)p(x∣H0​))]dx​

3.2 贝叶斯判决准则表达式

自然的，我们希望平均代价CCC是最小的。观察C的计算式，前两项c10P(H0)c_{10}P(H_0)c10​P(H0​)和c11P(H1)c_{11}P(H_1)c11​P(H1​)是大于0且无法改变的。但对于后面的那个积分，积分内的式子是两项相减的形式，所以要让平均代价最小，最好让积分就是负的。这就意味着

P(H1)(c01−c11)p(x∣H1)<P(H0)(c10−c00)p(x∣H0)P(H_1)(c_{01}- c_{11})p(x|H_1)<P(H_0)(c_{10}-c_{00})p(x|H_0) P(H1​)(c01​−c11​)p(x∣H1​)<P(H0​)(c10​−c00​)p(x∣H0​)

注意到积分是在R0R_0R0​上进行的，所以要让负值达到最大，应该尽可能让R0R_0R0​变大，那么让被积函数小于等于0的那些x值都应该划分给R0R_0R0​，那满足上述不等式时应该在R0R_0R0​中，即上述不等式应该是R0R_0R0​的判断，整理得到

H1:p(x∣H1)P(x∣H0)>P(H0)(c10−c00)P(H1)(c01−c11)H0:p(x∣H1)P(x∣H0)<P(H0)(c10−c00)P(H1)(c01−c11)\begin{aligned} H_1: \frac{p(x|H_1)}{P(x|H_0)} > \frac{P(H_0)(c_{10}-c_{00})}{P(H_1)(c_{01}-c_{11})} \\ H_0: \frac{p(x|H_1)}{P(x|H_0)} < \frac{P(H_0)(c_{10}-c_{00})}{P(H_1)(c_{01}-c_{11})} \end{aligned} H1​:P(x∣H0​)p(x∣H1​)​>P(H1​)(c01​−c11​)P(H0​)(c10​−c00​)​H0​:P(x∣H0​)p(x∣H1​)​<P(H1​)(c01​−c11​)P(H0​)(c10​−c00​)​​
