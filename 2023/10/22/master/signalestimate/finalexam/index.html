期末总结

1 经典的统计判决模型

1.1 基本统计检测模型

  1. 信源：发出信号的装置，对于二元信号来说，其发出两种信号，我们假设为H0H_0H0​与H1H_1H1​
  2. 概率转移机构：由于受到噪声干扰，概率转移机构将信源输出的信号以一定概率关系映射到观测空间。如果没有噪声干扰，信号在观测空间是确定的点
  3. 观测空间：观测信号可能取值的整个空间集合，例如二元信号取值的集合就是{H0,H1}\{H_0,H_1\}{H0​,H1​}
  4. 判决准则：在统计意义上作出判决的理论依据
  5. 判决规则：基于选定的判决准则并利用观测信号的先验信息对观测信号做出判决的机构

由于信号最终会落到观测空间中，所以统计判决的核心问题就是划分观测空间的问题。例如将空间中的一部分划分给H0H_0H0​，只要观测信号落到这部分，就判决为信号H0H_0H0​

1.2 对于判决结果的命名

以二元信号为例，两类判决正确的情况

  1. 假设H0H_0H0​，判决为H0H_0H0​，即P(H0∣H0)P(H_0|H_0)P(H0​∣H0​), 称为正确拒绝概率，记为PnP_nPn​
  2. 假设H1H_1H1​，判决为H1H_1H1​，即P(H1∣H1)P(H_1|H_1)P(H1​∣H1​), 称为检测概率，记为PdP_dPd​

两类判决错误的情况

  1. 假设H1H_1H1​，判决为H0H_0H0​，即P(H0∣H1)P(H_0|H_1)P(H0​∣H1​), 称为漏报概率，记为PmP_mPm​
  2. 假设H0H_0H0​，判决为H1H_1H1​，即P(H1∣H0)P(H_1|H_0)P(H1​∣H0​), 称为虚警概率，记为PfP_fPf​

在以上概率中存在以下关系

  1. Pf=1−PnP_f = 1-P_nPf​=1−Pn​
  2. Pd=1−PmP_d = 1-P_mPd​=1−Pm​

1.2 经典的判决模型

1.2.1 贝叶斯方法

我们的模型构造目标是通过构造一个判决模型，使得判决的结果最优。所以需要以一种最优的方式去划分观测空间。

为了用数学计算最优方法，引入判决的平均代价为

C=c00P(H0)P(H0∣H0)+c01P(H0)P(H0∣H1)+c10P(H1)P(H1∣H0)+c11P(H1)P(H1∣H1)\begin{aligned} C =& c_{00}P(H_0)P(H_0|H_0)+c_{01}P(H_0)P(H_0|H_1)+ \\ &c_{10}P(H_1)P(H_1|H_0)+c_{11}P(H_1)P(H_1|H_1) \end{aligned} C=​c00​P(H0​)P(H0​∣H0​)+c01​P(H0​)P(H0​∣H1​)+c10​P(H1​)P(H1​∣H0​)+c11​P(H1​)P(H1​∣H1​)​

其中cijc_{ij}cij​是做出这个判决需要付出的代价，这个式子的意义是使用代价cijc_{ij}cij​对四种判决的期望进行加权求和。

所以目标变成了使平均代价最小的判决模型，经过分析与推导，上式最小的时候满足

P(H1)(c01−c11)P(x∣H1)<P(H0)(c10−c00)P(H_1)(c_{01}-c_{11})P(x|H_1)<P(H_0)(c_{10}-c_{00}) P(H1​)(c01​−c11​)P(x∣H1​)<P(H0​)(c10​−c00​)

整理得到

P(x∣H1)P(x∣H0)≷H0H1P(H0)(c10−c00)P(H1)(c01−c11)\frac{P(x|H_1)}{P(x|H_0)}\stackrel{H_1}{\underset{H_0}{\gtrless}}\frac{P(H_0)(c_{10}-c_{00})}{P(H_1)(c_{01}-c_{11})} P(x∣H0​)P(x∣H1​)​H0​≷​H1​​P(H1​)(c01​−c11​)P(H0​)(c10​−c00​)​

即凡是使不等式小于0的都划分给判决H0H_0H0​的，因为要使平均代价最小化。这就是贝叶斯判决准则。其中

λ(x)=P(x∣H1)P(x∣H0)\lambda(x) = \frac{P(x|H_1)}{P(x|H_0)} λ(x)=P(x∣H0​)P(x∣H1​)​

被称为似然比函数。

η=P(H0)(c10−c00)P(H1)(c01−c11)\eta = \frac{P(H_0)(c_{10}-c_{00})}{P(H_1)(c_{01}-c_{11})} η=P(H1​)(c01​−c11​)P(H0​)(c10​−c00​)​

称作似然比检测门限。

贝叶斯判决准则的不等式通常可以化简，最终得到不等式左边是关于x的函数，右边是一个常数。又因为实际应用中我们经常会使用与指数函数相关的概率分布函数，所以两侧取对数会让公式变得更简单，取了对数之后的又叫做对数似然比

ln⁡λ(x)≷H0H1ln⁡η\ln \lambda(x) \stackrel{H_1}{\underset{H_0}{\gtrless}}\ln \eta lnλ(x)H0​≷​H1​​lnη

1.2.2 最小平均错误概率准则

这是一种经常用于通信系统的判决方法，其核心思想是c00=c11=0c_{00}=c_{11}=0c00​=c11​=0, c10=c01=1c_{10}=c_{01}=1c10​=c01​=1，即正确判断无代价，错误判断代价拉满。此时平均代价为

C=P(H0)P(H1∣H0)+P(H1)P(H0∣H1)C = P(H_0)P(H_1|H_0)+P(H_1)P(H_0|H_1) C=P(H0​)P(H1​∣H0​)+P(H1​)P(H0​∣H1​)

同样经过推导后，使得CCC最小使满足

λ(x)=P(x∣H1)P(x∣H0)≷H0H1P(H0)P(H1)=η\lambda(x) = \frac{P(x|H_1)}{P(x|H_0)}\stackrel{H_1}{\underset{H_0}{\gtrless}}\frac{P(H_0)}{P(H_1)} = \eta λ(x)=P(x∣H0​)P(x∣H1​)​H0​≷​H1​​P(H1​)P(H0​)​=η

特别的，当先验概率P(H0)=P(H1)P(H_0)=P(H_1)P(H0​)=P(H1​)，也就是接收的两种信号等概时，有

λ(x)=P(x∣H1)P(x∣H0)≷H0H11=η\lambda(x) = \frac{P(x|H_1)}{P(x|H_0)}\stackrel{H_1}{\underset{H_0}{\gtrless}}1 = \eta λ(x)=P(x∣H0​)P(x∣H1​)​H0​≷​H1​​1=η

1.2.3 最大后验概率准则

当代价因子c10−c00=c01−c11c_{10}-c_{00}=c_{01}-c_{11}c10​−c00​=c01​−c11​时，使得CCC最小的判决表达式是

P(H1)P(x∣H1)≷H0H1P(H0)P(x∣H0)P(H_1)P(x|H_1)\stackrel{H_1}{\underset{H_0}{\gtrless}}P(H_0)P(x|H_0) P(H1​)P(x∣H1​)H0​≷​H1​​P(H0​)P(x∣H0​)

又通过恒等变换推导，得到

P(H1∣x)≷H0H1P(H0∣x)P(H_1|x)\stackrel{H_1}{\underset{H_0}{\gtrless}}P(H_0|x) P(H1​∣x)H0​≷​H1​​P(H0​∣x)

注意到不等式变为了已获得观测量xxx的条件下，假设H0H_0H0​或H1H_1H1​为真的概率，是一种后验概率。

1.2.4 极小化极大准则

当不知道先验概率P(Hj)P(H_j)P(Hj​)，知道代价因子的时候，平均代价随先验概率变化，是有关于其中一种先验概率的函数，即

C=C(P1)C = C(P_1) C=C(P1​)

这个关于P1P_1P1​变化的函数被证明存在极大值点，而我们直接取C(P1)C(P_1)C(P1​)的极大值点作为我们猜测的P1P_1P1​，理由是取极大值点时，CCC将保持极大值不变，无论真实的P1P_1P1​如何，CCC的值都不会变。这个时候我们避免了产生过大代价。选取

∂C(P1,P1g)∂P1∣P1=P1g∗=0\frac{\partial C(P_1,P_{1g})}{\partial P_1}\vert_{P_1=P_{1g}^*}=0 ∂P1​∂C(P1​,P1g​)​∣P1​=P1g∗​​=0

得到

C(P1,P1g)=c00+(c10−c00)Pf(P1g)C(P_1, P_{1g}) = c_{00}+(c_{10}-c_{00})P_f(P_{1g}) C(P1​,P1g​)=c00​+(c10​−c00​)Pf​(P1g​)

如果进一步c00=c11=0c_{00}=c_{11}=0c00​=c11​=0, c10=c01=1c_{10}=c_{01}=1c10​=c01​=1

Pm(P1g∗)=Pf(P1g∗)P_m(P_{1g}^*) = P_f(P_{1g}^*) Pm​(P1g∗​)=Pf​(P1g∗​)

得到极小化极大的代价就是平均错误概率Pf(P1g∗)P_f(P_{1g}^*)Pf​(P1g∗​)
